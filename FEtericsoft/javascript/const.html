The const keyword was introduced in ES6 (2015). Variables defined with const cannot be Redeclared. Variables defined with const cannot be Reassigned. Variables defined with const have Block Scope.

It does not define a constant value. It defines a constant reference to a value. Because of this you can NOT: Reassign a constant value Reassign a constant array Reassign a constant object But you CAN: Change the elements of constant array Change the properties of constant object

If you want to learn more about hoisting, study the chapter JavaScript Hoisting. Variables defined with const are also hoisted to the top, but not initialized. Meaning: Using a const variable before it is declared will result in a ReferenceError:
